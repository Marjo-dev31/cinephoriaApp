# Ionic Angular Cinephoria App

[![Built with Ionic](https://img.shields.io/badge/-Built%20with%20Ionic-3880FF?style=flat&logo=ionic&logoColor=white)](https://ionicframework.com)
[![Angular](https://img.shields.io/badge/-Angular-DD0031?style=flat&logo=angular&logoColor=white)](https://angular.io)

This is **Cinephoria application** built with **Ionic Framework** and **Angular**. It allow user to access to their tickets for futures showings

## ⚙️ Getting Started

### Prerequisites

- Node.js (LTS version recommended)
- npm (included with Node.js)
- Ionic CLI → Install globally:

  ```bash
  npm install -g @ionic/cli
  ```

- Angular CLI → Install globally:

  ```bash
  npm install -g @angular/cli
  ```

### Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/Marjo-dev31/cinephoriaApp.git
    ```
2. Navigate into the project:
    ```bash
    cd cinephoriaApp
    ```
3. Install dependencies:

    ```bash
    npm install
    ```
4. Start the dev server:

    ```bash
    ionic serve
    ```
5. Open your browser to:

    ```
    http://localhost:8100
    ```

## Build

```
ionic build
```

## Serve on Android Studio

```
npx cap open android
```
